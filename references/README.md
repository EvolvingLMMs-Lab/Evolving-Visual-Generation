# References

This folder contains the BibTeX file used by the survey:

- [`citation.bib`](citation.bib)

The bibliography spans visual generation models, control methods, editing, unified multimodal systems, data construction, reward modeling, VLM-based evaluation, agentic generation, embodied generation, and world models.

## Suggested Contribution Format

When suggesting a missing paper, please include:

- title;
- authors;
- venue or arXiv link;
- year;
- BibTeX entry if available;
- the most relevant category, such as model architecture, training data, alignment, evaluation, editing, embodied generation, or world modeling.

Please use the GitHub issue template so new references can be triaged consistently.
