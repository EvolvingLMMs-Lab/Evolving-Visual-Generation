# Assets

This folder contains selected public-facing figures copied from the manuscript draft.

## Layout

- `figures/overview/`: taxonomy, publication trend, and research landscape figures.
- `figures/method/`: modeling paradigms, architecture, conditioning routes, and closed-source system loop figures.
- `figures/training/`: training pipeline, distillation paradigms, and data pipeline figures.
- `figures/stress-tests/`: selected qualitative stress-test examples.

Large or unfinished manuscript-only figures should stay out of the public repo until release cleanup. If new figures are added, prefer compressed PNG or SVG assets for README rendering and keep source-editable files in a separate design folder if needed.
