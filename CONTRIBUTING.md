# Contributing

This repository is an evolving survey and roadmap. Contributions are welcome in three forms:

1. **Missing papers:** suggest papers that fit the taxonomy or identify a gap in the bibliography.
2. **Taxonomy feedback:** point out unclear level boundaries, missing application regimes, or better terminology.
3. **Stress tests:** propose concrete prompts, input images, validators, or failure cases that probe higher-level visual intelligence.

## Paper Suggestions

Please include the paper title, year, URL, BibTeX if available, and a short note explaining where it fits:

- model and architecture;
- training, data, and alignment;
- conditional generation;
- image editing;
- unified multimodal generation;
- embodied generation and world models;
- evaluation and reward modeling.

## Style

- Keep summaries factual and concise.
- Prefer categorizing a paper by the capability it probes rather than only by the model family.
- For stress tests, describe the target invariant and how a failure can be verified.
- Do not add generated assets without noting the model, prompt, and settings when known.
